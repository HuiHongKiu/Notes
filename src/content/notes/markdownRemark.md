# Markdown Syntax Guide

## 📝 Headings
Use `#` symbols before your text:
```
# Heading 1
## Heading 2
### Heading 3
```
---
> # Heading 1
> ## Heading 2
> ### Heading 3


## 📄 Paragraphs & Line Breaks
Write text normally. Use two spaces at the end of a line for a line break.

## ✨ Emphasis
```
*italic* or _italic_
**bold** or __bold__
~~strikethrough~~
```
---
*italic* or _italic_

**bold** or __bold__

~~strikethrough~~

## 📋 Lists
### Unordered List
```
- Item 1
- Item 2
  - Subitem
```
---
- Item 1
- Item 2
  - Subitem

### Ordered List
```
1. First
2. Second
   1. Subitem
```
---
1. First
2. Second
   1. Subitem

## 🔗 Links
```
[Link text](https://example.com)
```
---
[Link text](https://example.com)

## 🖼️ Images
```
![Alt text](https://example.com/image.jpg)
```
---
![Alt text](https://example.com/image.jpg)

## 🧱 Code
```
    ```javascript

    ```
```
---
```javascript
function greet() {

```
---

```
- [x] Done task
- [ ] Pending task

Three or more *, -, or _:
```
---

- [ ] Pending task
